A Last.fm PHP library supporting many [API](http://www.lastfm.it/api) calls.

  * `album.getShouts`
  * `album.getTags`
  * `album.getTopTags`
  * `album.removeTag`
  * `album.search`
  * `album.share`
  * `artist.addTags`
  * `artist.getCorrection`
  * `artist.getEvents`
  * `artist.getImages`
  * `artist.getInfo`

  * `artist.getTags`
  * `artist.getTopAlbums`
  * `artist.getTopFans`
  * `artist.getTopTags`
  * `artist.getTopTracks`
  * `artist.removeTag`
  * `artist.search`
  * `artist.share`
  * `artist.shout`
  * `auth.getMobileSession`
  * `chart.getHypedArtists`

  * `chart.getTopTracks`
